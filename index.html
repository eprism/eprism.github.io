ePrism project

What's ePrism?

Small, cheap, efficient, and a programmable device with fascinating features for the price of a launch!

ePrism has 2.4" touch display, WiFi, audio output, +4MB fast flash storage, and you can program any kind of application you want and easily set it up inside of the device!

The main advantage of ePrism over the other choices like a tablet is the price. If you want to automate your services with a hardware for more than thousands of stores, clubs, classes & other places, you can use ePrism and save budget. Also, it's safer because the devices would be single purpose and very hard to misuse it.

Hardware

In Spite of the low cost, ePrism has a lot of features:

WiFi

Full 802.11 support. Use can use both STA and SoftAP operations. It means you can connect to WiFi or create an accesspoint for the other devices to connect to.

Bluetooth

ePrism fully supports low energy Bluetooth v4.2. You can use both Bluetooth and WiFi at same time.

LCD

Full-color high quality, touch 2.4" LCD helps you to create applications that you could just run on mobile phones before.

USB

You can use USB as a power source, or use it to program the device. Also, you can connect the other devices like PC through USB to ePrism.

Audio

High quality audio output. You can easily play audio file and notification in your applications.

Touch button

Accurate front touch button that enables users to interact with the device.

5GPIOs

5 Free GPIOs that makes you able to connect other devices or sensor easily.

Software

You can easily program ePrism device, just like your Android phone. Since the device is a single purpose, and the app would be the firmware itself, programming would be even easier than mobile phone programming. ePrism supports a wide range of programming languages, but because if simplicity, the official programming language is Arduino.

Unlike high-level platforms like Android and iOS, access the hardware is so simple and straightforward. You can use thousands of ready to use Arduino and C libraries to do anything you want. Also, there are too many free resource for learning the proramming language which is some cases it's even easier than mobile programming.

What you can do?

There is no limitation for software you can write, as long as you do not exceed the hardware limitation. You can:

  • Play WAV, MP3, AAC files
  • Play stream audio from internet
  • Show BMP, JPEG files
  • Use HTTP, SSL, Websocket, MQTT
  • And much more!
